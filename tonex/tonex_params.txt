METADATA BLOCK (19 parameters)
================================================================================
Preset identification and organizational tags

  GUID
    Type: varchar(32)
    Desc: Globally Unique Identifier for this preset
    Range: 32-character hex string

  Version
    Type: tinyint(10)
    Desc: Preset format version
    Range: Usually 3
    Notes: Default: 1

  ToneModel_GUID
    Type: varchar(32)
    Desc: Reference to the amp model tone capture
    Range: 32-character hex string

  OptionalToneModel_GUID
    Type: varchar(32)
    Desc: Optional secondary tone model for blending
    Range: 32-character hex string or empty

  Tag_PresetName
    Type: varchar(32)
    Desc: Display name of the preset (PRIMARY KEY)
    Range: Max 32 chars
    Notes: Must be unique

  Tag_UserName
    Type: varchar(32)
    Desc: Name of preset creator/user
    Range: Max 32 chars

  Tag_ModelerTags
    Type: varchar(32)
    Desc: Custom tags for organization
    Range: Max 32 chars

  Tag_Date
    Type: date
    Desc: Creation/modification date
    Range: ISO date format

  Tag_ModelCategory
    Type: varchar(32)
    Desc: Amp category classification
    Range: CLEAN, DRIVE, CRUNCH, FUZZY, etc.

  Tag_Instrument
    Type: varchar(32)

  Tag_InstrumentType
    Type: varchar(32)

  Tag_PickupPosition
    Type: varchar(32)
    Desc: Guitar pickup selector position
    Range: Bridge, Neck, Middle, None

  Tag_PickupType
    Type: varchar(32)
    Desc: Pickup configuration

  Tag_Artist

    Range: Max 32 chars

  Tag_Album
    Type: varchar(32)

    Range: Max 32 chars

  Tag_Song

    Range: Verse, Chorus, Solo, etc.

  Tag_Genre

  Tag_Description

================================================================================
AMP MODEL BLOCK (7 parameters)
================================================================================
Core amplifier tone and gain staging controls

  ModelEnable
    Type: tinyint(10)
    Desc: Enable/bypass the amp model
    Range: 0=off, 1=on

  ModelMix
    Type: float(24)
    Desc: Wet/dry blend of processed signal
    Range: 0.0-100.0%
    Notes: 100% = fully processed tone

  ModelGain
    Type: float(24)
    Desc: Input gain/drive amount
    Range: 0.0-10.0
    Notes: Controls amp saturation/distortion

  ModelVolume
    Type: float(24)
    Desc: Output level/master volume
    Range: 0.0-10.0
    Notes: Final amp output level

  PwrAmpEqPresence
    Type: float(24)
    Desc: Power amp presence control (high-end clarity)
    Range: 0.0-10.0
    Notes: Adds brightness and air

  PwrAmpEqDepth
    Type: float(24)
    Desc: Power amp resonance/depth (low-end)
    Range: 0.0-10.0
    Notes: Adds bass response and thump

  BPM
    Type: float(24)
    Desc: Tempo reference for time-based effects
    Range: 20.0-999.0 BPM
    Notes: Used for sync'ed delays/modulation

EQ BLOCK (8 parameters)
================================================================================
3-band parametric equalizer with frequency control

  EqPost
    Type: tinyint(10)
    Desc: EQ placement in signal chain
    Range: 0=pre-amp, 1=post-amp
    Notes: Affects tonal character significantly

  EqBass
    Type: float(24)
    Desc: Bass boost/cut amount
    Range: -10.0 to +10.0 dB
    Notes: Low frequency control

  EqBassFreq
    Type: float(24)
    Desc: Bass center frequency
    Range: 20.0-2000.0 Hz
    Notes: Typical: 80-400 Hz

  EqMid
    Type: float(24)
    Desc: Midrange boost/cut amount
    Range: -10.0 to +10.0 dB
    Notes: Most critical for tone shaping

  EqMidQ
    Type: float(24)
    Desc: Midrange bandwidth/Q factor
    Range: 0.1-10.0
    Notes: Lower Q = wider, Higher Q = narrower

  EqMidFreq
    Type: float(24)
    Desc: Midrange center frequency
    Range: 200.0-5000.0 Hz
    Notes: Typical: 500-2000 Hz

  EqTreble
    Type: float(24)
    Desc: Treble boost/cut amount
    Range: -10.0 to +10.0 dB
    Notes: High frequency control

  EqTrebleFreq
    Type: float(24)
    Desc: Treble center frequency
    Range: 1000.0-20000.0 Hz
    Notes: Typical: 2000-8000 Hz
